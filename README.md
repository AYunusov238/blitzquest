A Django-based multiplayer board game project.

---

## Overview
**BlitzQuest** is a turn-based board game built with Django where players move across tiles, answer questions, face traps, earn bonuses, and compete to win under different game modes.  
The project is designed for collaborative development and easy local setup.

---

## Tech Stack
- Python 3.x  
- Django  
- HTML / CSS / JavaScript  
- SQLite (default, for development)
